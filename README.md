# hundredhires-portfolio

## Development Environment Setup

### Tools Installed

- **Cursor IDE** — AI-powered code editor used as the primary development environment
- **Claude Code extension** — AI coding assistant integrated into Cursor
- **Codex extension** — Additional AI coding assistant for pair programming and code generation

### Setup Steps Completed

1. Created a GitHub repository for this project
2. Cloned the repository locally
3. Installed and configured Cursor IDE
4. Set up both AI coding assistants (Claude Code and Codex extensions)

### Issues Encountered & Resolutions

| Issue | Resolution |
|-------|------------|
| Git was not installed on the system | Installed [Git for Windows](https://git-scm.com/download/win) |
| Could not locate the Extensions panel in Cursor | Opened it via the Command Palette (`Ctrl+Shift+P` → "Extensions: Install Extensions") |
